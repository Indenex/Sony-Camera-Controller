This application is only designed to run on Firefox version 65/66, or Chrome with some exceptions*. It's very likely the xorigin workarounds available in these versions of Firefox will be deprecated at some point. 


Caveats: 
If camera usernames/passwords are updated in the camera configurations, you will need to update the usernames/passwords within the source file's javascript as well. 
This application is not compatible with MPEG4 livestreaming. 
This program is very insecure as the IP camera hardware I developed it for came out circa 2006, and was itself **very** insecure. :)  







* with command line edits located here, not enabled by default: https://stackoverflow.com/questions/45039335/bypass-blocking-of-subresource-requests-whose-urls-contain-embedded-credentials
